# Helpdesk / CRM Integration

1. Choose your system:
   - HubSpot, Zendesk, GHL, Freshdesk, Airtable, etc.

2. When a conversation is escalated:
   - Create a ticket or contact record.
   - Attach:
     - Name
     - Email / Phone
     - Conversation summary
     - Tags (e.g., URGENT, BILLING, TECHNICAL)

3. Use Zapier / Make:
   - Trigger on "escalation tag" or custom webhook.
   - Create ticket in helpdesk.
   - Notify support team via email/Slack/SMS.
